# SMS Auth

API for passwordless SMS authentication powered by Twilio. Features include session management, account lock (too many attempts), account recovery, IP tracking, anonymous login, and more.

`POST /sms/signup`

`POST /sms/login`

`POST /sms/verify`

`POST /anon/signup`

`POST /anon/login`

`POST /backup/login`

`POST /session/refresh`

`POST /session/logout`
